# Hospital-Data-Analysis
 Hospital Management Dashboard — Power BI Project
An interactive, multi-page Power BI dashboard designed to monitor hospital operations including patient records, doctor performance, bed & medicine stock status, and financial analytics.

📌 Project Overview
This Power BI project provides a comprehensive Hospital Management Analytics Dashboard that enables hospital administrators to track and analyze key operational metrics in real time. The dashboard is organized across 6 dedicated report pages, each focused on a specific domain of hospital management.

Built as part of my learning journey in Business Intelligence and Data Visualization, this project demonstrates real-world application of Power BI features including DAX measures, slicers, drill-through navigation, KPI cards, and dynamic visuals.

📂 Report Pages
#	Page Name	Description
1	Home	Landing/navigation page with buttons linking to all report sections
2	Overview	High-level KPIs — Total Patients, Staff Count, Doctors, Revenue, Bed Status, Medicine Stock
3	Patient	Individual patient drill-down with charges breakdown, medicines bought, and health metrics
4	Doctor	Doctor-wise performance, commission gauge, appointment schedule, and specialty filters
5	Hospital	Operational view — patient age category distribution, bed occupancy, test status, appointments
6	Finance	Revenue trends, charge type analysis, and financial KPI cards
📊 Key Visuals Used
KPI Cards — Total Patients, Doctors, Staff Count, Revenue/Amount
Donut Charts — Bed Status, Medicine Stock Status
Clustered Bar Charts — Charges by Type, Medicines Bought per Patient
Stacked Area Chart — Patient trends over time
100% Stacked Column/Bar Charts — Bed occupancy %, Charge distribution
Gauge Chart — Doctor commission tracking
Pivot/Matrix Tables — Medicine inventory, Doctor appointments, Patient test status
Slicers — Patient name, Doctor name, Date range, Department
Page Navigator — Seamless navigation between report pages
Action Buttons — Interactive Edit/Close panel toggles
